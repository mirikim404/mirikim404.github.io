## 1. 정규식이란?

문자열의 **패턴을 검사/추출**하기 위한 표현식.
코테에서 주로 `String.matches()`와 함께 씀.

---

## 2. 자주 쓰는 패턴 기호

| 기호 | 의미 | 예시 |
|------|------|------|
| `[0-9]` | 숫자 한 글자 | `"3".matches("[0-9]")` → `true` |
| `[a-z]` | 소문자 한 글자 | `"a".matches("[a-z]")` → `true` |
| `[a-zA-Z]` | 영문자 한 글자 | |
| `\d` | 숫자 (`[0-9]`와 동일) | Java에서는 `\\d` |
| `\w` | 숫자+영문+언더스코어 | Java에서는 `\\w` |
| `.` | 임의의 문자 한 글자 | |
| `*` | 0번 이상 반복 | `"".matches("[0-9]*")` → `true` |
| `+` | 1번 이상 반복 | `"".matches("[0-9]+")` → `false` |
| `?` | 0번 또는 1번 | |
| `{n}` | 정확히 n번 반복 | `"1234".matches("[0-9]{4}")` → `true` |
| `{n,m}` | n번 이상 m번 이하 | |
| `^` | 문자열 시작 | `find()` 쓸 때 필요 |
| `$` | 문자열 끝 | `find()` 쓸 때 필요 |
| `\|` | 또는 (OR) | `"1234".matches("[0-9]{4}\|[0-9]{6}")` → `true` |

> Java에서 `\d`, `\w` 등은 백슬래시를 이스케이프해야 해서 `\\d`, `\\w`로 씀!

---

## 3. `matches()` 사용법

```java
String s = "1234";

// 전체가 숫자인지
s.matches("[0-9]+");      // true
s.matches("\\d+");        // true (동일)

// 정확히 4자리 또는 6자리 숫자인지
s.matches("[0-9]{4}|[0-9]{6}");  // true

// 영문자만
"hello".matches("[a-zA-Z]+");    // true

// 이메일 형식
"test@gmail.com".matches("[a-zA-Z0-9]+@[a-zA-Z]+\\.[a-zA-Z]+"); // true
```

### `matches()` 특징
- 문자열 **전체**가 패턴과 일치해야 `true`
- 그래서 `^$` 없이 써도 전체 매칭으로 동작함

```java
"1234abc".matches("[0-9]{4}");  // false (전체 매칭이라 abc 때문에 탈락)
```

---

## 4. `matches()` vs `find()`

부분 매칭이 필요하면 `Pattern` + `Matcher` 사용.

```java
import java.util.regex.*;

String s = "1234abc";

// matches() - 전체 매칭
s.matches("[0-9]{4}");                              // false

// find() - 부분 매칭
Pattern.compile("[0-9]{4}").matcher(s).find();      // true ← 앞 4자리만 찾음

// find()에서 전체 매칭 강제하려면 ^$ 사용
Pattern.compile("^[0-9]{4}$").matcher(s).find();   // false
```

> 코테에서는 대부분 `matches()`로 충분!

---

## 5. 코테 활용 예시

```java
// 숫자로만 이루어진 4자리 또는 6자리 문자열 검사
public boolean solution(String s) {
    return s.matches("[0-9]{4}|[0-9]{6}");
}

// 알파벳 소문자로만 이루어졌는지
public boolean isLower(String s) {
    return s.matches("[a-z]+");
}

// 숫자를 문자열로 변환 후 사용
String.valueOf(12345).matches("[0-9]{5}");  // true
```

## 정리 표

| 목적 | 패턴 | 예시 |
|------|------|------|
| 전체 숫자 | `[0-9]+` 또는 `\\d+` | `"123".matches("\\d+")` |
| n자리 숫자 | `[0-9]{n}` | `"1234".matches("[0-9]{4}")` |
| n자리 or m자리 | `[0-9]{n}\|[0-9]{m}` | `"123456".matches("[0-9]{4}\|[0-9]{6}")` |
| 영문자만 | `[a-zA-Z]+` | `"abc".matches("[a-zA-Z]+")` |
| 영문+숫자 | `[a-zA-Z0-9]+` | |
| 임의 n글자 | `.{n}` | `"abc".matches(".{3}")` |
